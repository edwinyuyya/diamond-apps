📋 Langkah Deploy (±20 Menit)
Ikuti urutan ini dari atas ke bawah
1
Buat Akun GitHub
GitHub adalah tempat menyimpan kode sebelum di-deploy ke Vercel. Gratis selamanya.
🔗 Langkah:
1. Buka github.com
2. Klik Sign Up → isi email & password
3. Verifikasi email
4. Selesai! ✅
2
Upload File ke GitHub
Buat repository baru dan upload semua file yang sudah didownload.
📤 Langkah:
1. Klik tombol + di GitHub → New repository
2. Nama repo: mua-poster
3. Pilih Public → klik Create repository
4. Klik uploading an existing file
5. Drag semua file yang didownload → klik Commit changes
⚠️ Perhatian: Jangan upload file .env yang berisi API key! Hanya upload .env.example saja.
3
Setup Supabase (Database)
Supabase adalah database gratis untuk menyimpan data akun MUA dan jadwal post.
🗄️ Langkah:
1. Buka supabase.com → Sign Up (bisa pakai GitHub)
2. Klik New Project → isi nama: mua-poster
3. Pilih region: Southeast Asia (Singapore)
4. Buat password database (catat!)
5. Tunggu ~2 menit sampai project siap
6. Buka SQL Editor → klik New Query
7. Copy isi file schema.sql → paste → klik Run
8. Buka Project Settings → API
9. Catat: Project URL & anon public key
4
Deploy ke Vercel
Vercel akan hosting app dan menjalankan scheduler otomatis setiap menit.
🚀 Langkah:
1. Buka vercel.com → Sign Up with GitHub
2. Klik Add New → Project
3. Pilih repo mua-poster → klik Import
4. Di bagian Environment Variables, tambahkan:

  • NEXT_PUBLIC_SUPABASE_URL = URL dari Supabase
  • NEXT_PUBLIC_SUPABASE_ANON_KEY = anon key Supabase
  • CRON_SECRET = buat password bebas (contoh: MUAPoster2024!)
  • ANTHROPIC_API_KEY = API key dari console.anthropic.com

5. Klik Deploy → tunggu ~3 menit
6. App online! Dapat URL seperti: mua-poster.vercel.app
5
Aktifkan Cron Job di Vercel
Ini yang bikin posting jalan otomatis tanpa HP harus nyala!
⏰ Langkah:
1. Di Vercel dashboard → buka project mua-poster
2. Klik tab Settings → Cron Jobs
3. Seharusnya sudah muncul cron dari vercel.json
4. Tambahkan environment variable:
   CRON_SECRET = sama dengan yang diisi sebelumnya
5. ✅ Cron aktif! Akan cek jadwal setiap 1 menit
✅ Cek berhasil: Buka URL https://mua-poster.vercel.app/api/cron — kalau muncul response JSON, berarti berjalan!
6
Bagikan ke Tim MUA
Setelah deploy, semua anggota tim bisa langsung pakai dari HP masing-masing!
📱 Cara Akses:
1. Bagikan URL app ke semua anggota tim:
   https://mua-poster.vercel.app
2. Setiap anggota bisa buka dari HP masing-masing
3. Akun MUA hanya perlu diinput sekali (tersimpan di database)
4. Bisa langsung mulai upload & jadwalkan post! 🎉
📸 Setup Instagram API
Untuk setiap akun MUA, perlu Access Token dari Facebook Developer
⚠️ Syarat: Akun Instagram harus Business atau Creator, terhubung ke Facebook Page.
LangkahYang Dilakukan
1 Bukadevelopers.facebook.com → Login dengan Facebook
2 Buat AppMy Apps → Create App → pilih Other → Business
3 Tambah ProdukDashboard → Add Product → Instagram Graph API
4 Generate TokenTools → Graph API Explorer → pilih App → Generate Token
5 Set PermissionCentang: instagram_basic, instagram_content_publish, pages_read_engagement
6 Dapat User IDDi Graph Explorer, jalankan query /me → catat field id
7 SelesaiInput User ID + Token di tab Akun MUA di app
💡 Tips: Token default hanya berlaku 1 jam. Untuk token jangka panjang (60 hari), gunakan Long-Lived Token. Cari tutorial "extend Instagram access token 60 days" di YouTube.

❓ Pertanyaan Umum
Yang sering ditanyakan
❓ Kalau HP dimatikan, jadwal tetap jalan?
✅ Ya! Setelah deploy ke Vercel, jadwal jalan otomatis di cloud. HP boleh dimatikan, app boleh ditutup.
❓ Apakah 10 anggota tim bisa akses bersamaan?
✅ Ya! Semua anggota cukup buka URL app dari browser HP masing-masing. Bisa online bersamaan.
❓ Data akun aman tidak?
✅ Data tersimpan di Supabase dengan enkripsi. Access Token bersifat sensitif — jangan bagikan ke orang lain.
❓ Kalau token expired, gimana?
⚠️ Token Instagram berlaku 60 hari (long-lived). Perlu diperbarui manual tiap 60 hari di tab Akun MUA.
